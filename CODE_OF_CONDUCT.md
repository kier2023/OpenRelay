# Code of Conduct

## Purpose

This project is maintained by a single developer. Contributions, feedback, and discussion are welcome, but they must remain respectful and constructive.

The goal is simple: build useful tools and share them openly. That works best when discussions remain focused on improving the software rather than attacking people.

---

## Expected Behavior

Anyone participating in this project is expected to:

- Communicate respectfully in issues, pull requests, and discussions
- Provide constructive and technically focused feedback
- Stay on topic when discussing features or problems
- Accept that not every suggestion or pull request will be accepted
- Respect the time and decisions of the maintainer

Disagreements about technical decisions are normal. They should be handled through evidence, reasoning, and calm discussion.

---

## Unacceptable Behavior

The following behavior will not be tolerated in issues, pull requests, discussions, or any other project spaces:

- Harassment or personal attacks
- Discriminatory language or hate speech
- Threats, intimidation, or abusive behavior
- Spam, trolling, or intentionally disruptive comments
- Repeatedly ignoring project guidelines after being asked to stop

If something wouldn’t be appropriate in a professional technical discussion, it likely doesn’t belong here.

---

## Enforcement

This project is maintained by a single maintainer. Moderation decisions are made solely by the maintainer.

If unacceptable behavior occurs, the maintainer may:

- Remove comments
- Close issues or pull requests
- Block or ban users from participating in the project

These actions may occur without warning if behavior is clearly disruptive or abusive.

---

## Reporting Issues

If you encounter behavior that violates this Code of Conduct, you can report it by opening an issue or contacting the maintainer through the project's repository.

Reports will be reviewed and handled at the maintainer’s discretion.

## Final Note

Open source works best when people collaborate in good faith. Respectful discussion, thoughtful contributions, and constructive criticism are always welcome.
